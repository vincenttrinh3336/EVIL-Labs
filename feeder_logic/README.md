# Pet Feeder python scripts handling all sensor, computer vision, and backend logic


## Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd pet-feeder-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   pip install fastapi uvicorn
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
